Instructions

Create Node: Click on a blank space in the canvas (above) to create a new node.

Create Edge: Drag from source node to destination node to create a edge between them. (For non-communative effects like Dm where direction matters, the direction of the edge is from source to destination.)

Modify Node or Edge Parameters: Select a node or edge by clicking on it. The current parameters will be displayed aside (or below) the canvas in the "form" section. These parameters can be modified by editing the fields in the form. The new values will be immediately saved and applied.

Delete Node: Drag node to the red "X", then drop.

Delete Edge: Ctrl-click on an edge to delete it. Note: edges will automatically be deleted if one of their attached nodes is deleted.

Move Node: You can move the location of a node, by dragging it to an unoccupied location. Note: moving a node, will also move its edges appropriately.
